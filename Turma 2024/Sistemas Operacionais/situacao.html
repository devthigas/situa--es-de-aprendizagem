2 - Estratégia de aprendizagem desafiadora

Contextualização

Você foi contratado para desenvolver, em colaboração com a equipe de UX Design, um painel administrativo genérico que poderá ser comercializado e adaptado para diferentes necessidades. Esse painel servirá como base para diversas aplicações, permitindo que os compradores personalizem e integrem facilmente em seus projetos.

Antes do desenvolvimento, a equipe de UX realizou uma pesquisa para entender as necessidades dos usuários e criou um protótipo de baixa fidelidade, que servirá como referência para o design e a experiência do painel.

A equipe definiu que o painel será desenvolvido com Tailwind CSS, pois esse framework agiliza o desenvolvimento, proporciona maior flexibilidade na personalização e facilita a adaptação do painel pelos futuros compradores.

Objetivo

Desenvolver um painel administrativo responsivo, utilizando Tailwind CSS, que seja flexível, intuitivo e esteticamente agradável. O painel deverá seguir boas práticas de UI e UX Design, garantindo interatividade e usabilidade para diferentes cenários comerciais.

Desafio

Criar um painel genérico, que possa ser utilizado em diversas situações, garantindo uma interface interativa e responsiva. Além disso, é necessário estruturar o código de forma organizada, permitindo personalizações fáceis para diferentes clientes.

O painel deve incluir:

  • Implementação do protótipo de baixa fidelidade desenvolvido pela equipe de UX, garantindo que a interface atenda às necessidades identificadas na pesquisa.
  • Modais interativos para exibição de informações ou confirmação de ações.
  • Menus dinâmicos, que se adaptem a diferentes tamanhos de tela e melhorem a navegação.
  • Animações sutis, para melhorar a experiência do usuário sem comprometer o desempenho.

Resultado Esperado

Ao final do desenvolvimento, espera-se obter um painel administrativo funcional, responsivo e pronto para ser comercializado, com:

  • ✅ Interface moderna e intuitiva, baseada no protótipo de baixa fidelidade criado pela equipe de UX.
  • ✅ Código modular e bem documentado, facilitando a personalização e manutenção.
  • ✅ Experiência fluida para o usuário, garantindo acessibilidade e interatividade em diferentes dispositivos.

Esse projeto permitirá aprimorar suas habilidades no desenvolvimento de interfaces dinâmicas e na aplicação de conceitos avançados de responsividade e usabilidade.
